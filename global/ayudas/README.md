# Ayudas para parciales de Programación Funcional

Índice de la guía, separada en archivos por tema.

- [recordatorios.md](recordatorios.md) — qué hacer antes de escribir código, y checklist final antes de entregar un ejercicio.
- [esquemas.md](esquemas.md) — cuándo hace falta `fold`/`rec`, alcance real de "sin recursión explícita".
- [patrones.md](patrones.md) — formas recurrentes de los ejercicios (lista disfrazada, ramas simultáneas, ramas secuenciales, ramificación con dos campos, simplificar/colapsar, fusionar listas ordenadas, contar pasos).
- [demos.md](demos.md) — estructura de una demostración: las 3 columnas, plantilla completa con las frases exactas, extensionalidad, uso de HI/lemas, demo por casos, demo con `if`.
- [funciones.md](funciones.md) — tabla de funciones de Prelude/combinadores usados seguido (tipo + para qué sirve).
- [sintaxis.md](sintaxis.md) — `let`/`where`/`case`/guards/lambda/secciones.

- [../demos-template.hs](../demos-template.hs) — esqueleto de demo en formato `.hs`.
